## How to Cite

If you use the Modernized Biomarker Discovery Framework in academic research, please cite the repository and any associated publications.

A formal citation entry and DOI (via Zenodo) will be provided here once assigned as part of the Version 1.0 release.

---

## Interim Citation

Until a formal citation and DOI are available, please reference the GitHub repository directly:

```
Tran, N. (2026). Modernized Biomarker Discovery Framework [Computer software].
GitHub. https://github.com/<username>/ModernizedBiomarkerDiscoveryFramework
```

Please replace the URL with the canonical repository URL at the time of citation.

---

## Citing the Demonstration Case

If your work specifically references the 12-gene gastric cancer signature demonstration (Part 1 and/or Part 2's application of it), please cite both the framework (above) and the original published signature study that Part 1 reconstructs, in accordance with standard academic attribution practice.
